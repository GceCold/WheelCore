IndusrialWorld是一个Bukkit/Spigot服务器插件，旨在为Minecraft添加像Mod一样的效果。  
提供更多的矿物，机器；酷炫的武器和强大的盔甲。  
一个崭新的玩Minecraft的方法！ 
不需要任何服务器Mod！ 

需要Spigot 1.13.2(理论支持1.13.2及以上)服务器以运行。
