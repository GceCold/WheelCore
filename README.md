# IndustrialWorld 
~~大扎好，我系渣渣辉，IndustrialWorld，介四里没有挽过的船新版本，挤需体验三番钟，里造会干我一样，爱象节个插件。~~  

IndusrialWorld是一个Bukkit/Spigot服务器插件，旨在为Minecraft添加像Mod一样的效果。  
提供更多的矿物，机器；酷炫的武器和强大的盔甲。  
一个崭新的玩Minecraft的方法！ 
不需要任何服务器Mod！
